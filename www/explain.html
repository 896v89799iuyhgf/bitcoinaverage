First we collect data from all available exchanges over the Internet. Data consists of a) nominated currency b) last trade price, c) current lowest ask, d) current highest bid, e) 24h sliding window trading volume. For multicurrency exchanges like mtgox we collect multiple such sets, separately for each currency.
This data is collected approximately every minute, which may slightly vary depending on the load time for various exchanges

When data is collected we calculate weighted averages within each currency using trading volumes as weights. Weighted averages are calculated for last price, bid and ask separately.

These weighted averages are shown on site as currency markets data.

Of all globally recognized currencies only around a dozen are shown in the market area. This is because we recognize as markets only currencies that are directly traded to BTC anywhere in the world.

Global average is calculated using fiat exchange rates, so it can be shown in nearly any fiat currency. To calculate global average we choose a target currency (USD as example). Then we convert each market's weighted average into target currency using fiat exchange rates provided by openexchangerates.org.

Finally global weighted average is calculated using all market's averages together, converted into target currency. Each market's total trading volume is used as weight for calculation.
